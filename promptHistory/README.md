# Prompt History - Agent Chat Logs

This folder contains all agent chat transcripts from Cursor AI sessions for this project.

## Files

All files are agent transcripts saved from Cursor's internal storage on February 16, 2026.

Each file is named with a unique identifier (UUID) and contains the complete conversation history between the user and the AI agent for that session. **Each markdown file includes timestamp metadata** in the YAML frontmatter showing when the conversation occurred.

## Viewing Transcripts

### Web Interface
Open `../prompt-history.html` in your browser to view all transcripts with:
- Timestamps and dates for each session
- Sortable cards (newest first, oldest first, largest first)
- Click any card to view the full transcript
- Session statistics and size information

### Direct File Access
All transcript files include metadata headers with:
- Agent session ID
- Timestamp (YYYY-MM-DD HH:MM:SS)
- Formatted date and time
- ISO 8601 timestamp
